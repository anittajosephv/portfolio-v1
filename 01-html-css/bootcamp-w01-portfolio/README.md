# Anitta Joseph Portfolio

## Description

This is my first portfolio project built as part of the Fullstack Developer Bootcamp. It demonstrates semantic HTML5, accessible forms, and minimal CSS styling.

## Features

- Semantic HTML sections (header, main, sections, footer)
- Contact form with accessible labels
- Minimal CSS styling for layout and typography

## How to Run Locally

1. Clone the repository: https://github.com/anittajosephv/fullstack-training.git
2. Open `index.html` in your browser.
